Використання lemma_text не дало суттєвого покращення baseline, а в частині випадків якість навіть знизилась. Лематизація не допомогла у випадках синонімічних конструкцій («Бога немає» — «Бога не існує», «Я не заперечую» — «Я не проти»), оскільки TF-IDF не враховує глибинну семантику. Для нашого проєкту доцільно не використовувати леми як основне представлення для класифікації
